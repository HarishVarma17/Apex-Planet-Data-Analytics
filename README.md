# TASK-1 : Data Immersion & Wrangling

## Objective

This task focuses on understanding the dataset, identifying data quality issues, cleaning the data, performing feature engineering, and preparing an analysis-ready dataset.

---

# Dataset Overview

The dataset contains Superstore sales transaction details including:

* Customer Information
* Product Information
* Sales Details
* Shipping Details
* Profit & Discount Information

Total Records: **9994 Rows**
Total Columns: **21 Columns**

---

# Operations Performed

## Understanding Dataset Structure

* Analyzed dataset columns and data types
* Checked dataset dimensions
* Explored categorical and numerical features

---

## Data Quality Checks

Performed multiple quality assessment operations:

* Missing Value Detection
* Duplicate Record Detection
* Outlier Identification
* Data Type Verification

---

## Data Cleaning

The following cleaning operations were performed:

* Removed duplicate rows
* Standardized column names
* Converted date columns into datetime format
* Verified null values

---

## Feature Engineering

Created new meaningful features for better analysis:

### Engineered Features

* **Customer Age**
* **Total Revenue**
* **Transaction Month**
* **Delivery Days**
* **Profit Category**
